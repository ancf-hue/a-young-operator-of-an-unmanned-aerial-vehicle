## 📖 Описание проекта

**«Юный оператор БПЛА»** — это инновационная образовательная платформа, которая превращает обучение управлению дронами в увлекательное приключение! 

Курс разработан специально для детей и подростков, которые хотят освоить профессию оператора беспилотных летательных аппаратов. Через интерактивные симуляторы и игровые механики ученики проходят путь от первого взлёта до сложных спасательных операций.

### 🎯 Цели курса:
- ✅ Научить основам пилотирования дронов
- ✅ Развить пространственное мышление и координацию
- ✅ Изучить технику безопасности при полётах
- ✅ Подготовить к реальной работе с БПЛА
- ✅ Вдохновить на карьеру в авиации и робототехнике

## 🌟 Особенности проекта

### 🎨 **Современный UI/UX дизайн**
- **Glassmorphism 3D** — стеклянный интерфейс с эффектом глубины
- **Анимированные кнопки** с переливающимся градиентом и 3D-эффектом
- **Адаптивная сетка** — 5 уроков в ряду (адаптация под все устройства)
- **Пульсирующий логотип** с неоновым свечением
- **Тёмная тема** с киберпанк-эстетикой

### 📊 **Система прогресса**
- Визуальный прогресс-бар прохождения курса
- Отметка выполненных уроков (сохранение в localStorage)
- Мотивационная система достижений

### 🎮 **25 интерактивных симуляторов**
Каждый урок — это отдельный симулятор с уникальной миссией:
- От базовых навыков (взлёт, посадка, зависание)
- До продвинутых техник (полёт в ветреную погоду, отказ двигателя, групповой полёт)

## 📚 Программа курса

### **Базовый уровень (Уроки 1-7)**
1. 🎯 Знакомство с дроном и взлёт
2. ⭕ Зависание на высоте
3. ⬆️️ Движение вперёд и назад
4. ⬅️➡️ Движение влево и вправо
5. 🔄 Повороты на месте
6. ⬜ Полёт по квадрату
7. 🎯 Посадка на мишень

### **Средний уровень (Уроки 8-14)**
8. 🌳 Облёт статичных препятствий
9. ⭕ Полёт сквозь кольцо
10. 8️⃣ Восьмёрка
11. 💨 Действие ветра
12. 📷 Работа с камерой
13. 🌙 Ночной полёт
14. 📟 Полёт по приборам

### **Продвинутый уровень (Уроки 15-21)**
15. 🚨 Экстренная посадка
16. 📡 Потеря сигнала (RTH)
17. 🎬 Съёмка движущегося объекта
18. 🏠 Полёт в тесном пространстве
19.  Основы аэродинамики
20. 🔧 Обслуживание дрона
21. 🐝 Групповой полёт (Рой)

### **Экспертный уровень (Уроки 22-25)**
22. 🗺️ Картография местности
23. ⚠️ Симулятор отказа мотора
24. 🏆 Прохождение полосы препятствий
25. 🆘 Финальный экзамен «Спасательная операция»

## 🛠 Технологии

### **Frontend**
- **HTML5** — семантическая разметка
- **CSS3** — современные стили с анимациями
  - Glassmorphism эффекты (`backdrop-filter`)
  - CSS Grid (5 колонок)
  - CSS Animations & Keyframes
  - CSS Variables (кастомные свойства)
  - Media Queries (адаптивность)
- **JavaScript (ES6+)** — интерактивность
  - LocalStorage API (сохранение прогресса)
  - Dynamic DOM manipulation
  - Event handling

### **Дизайн**
- **Цветовая палитра**: Cyberpunk/Cyan-Purple gradient
- **Шрифты**: Segoe UI, system fonts
- **Иконки**: Favicon.ico + Emoji
- **Эффекты**: 
  - 3D кнопки с перспективой
  - Переливающийся градиент (анимация)
  - Неоновое свечение (box-shadow)
  - Пульсация логотипа
